# Thanuja Sekuri — Portfolio Airport

An interactive airport-themed software engineering portfolio showcasing my projects, technical skills, experience, education, research, and achievements.

## Live website

### [Visit Thanuja's Portfolio Airport](https://thanuja-portfolio-delta.vercel.app/)

## About the portfolio

Visitors navigate the website through an airport departure board. Each destination opens a different part of my journey, while boarding passes, passport stamps, flight transitions, education galleries, and optional airplane ambience make the experience more engaging.

> **Boarding all ideas.**  
> ***Destination: production.***  
> Every flight I've taken was a problem worth solving.

## Features

- Interactive airport departure board
- Education, skills, projects, experience, achievements, and contact sections
- Boarding-pass navigation and passport progress
- One-second airplane transition between new destinations
- Optional airplane ambience, off by default
- Keyboard and on-screen movement controls
- Expandable education photo galleries
- Certificate previews
- Responsive layout for desktop and mobile
- Direct GitHub and live-demo links for projects

## Run locally

Requirements: Node.js 22 or newer and npm.

```bash
git clone https://github.com/thanujaa9/Thanuja-Portfolio.git
cd Thanuja-Portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production build

```bash
npm run build
npm start
```

## Deployment

The portfolio is deployed on Vercel at [thanuja-portfolio-delta.vercel.app](https://thanuja-portfolio-delta.vercel.app/).

To create another deployment, import the GitHub repository, select the Next.js preset, keep the root directory as `./`, and use the default build settings.

Every push to the `main` branch triggers a new production deployment after the Vercel project is connected.
